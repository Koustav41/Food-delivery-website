# 🍕 Food Delivery Website

A modern, feature-rich food delivery web application built with JavaScript, HTML, and Python. This project enables users to browse restaurants, order food, and track deliveries seamlessly.

## ✨ Features

- 🏪 **Browse Restaurants** - Explore a variety of restaurants and cuisines
- 🍔 **Menu Browsing** - View detailed menus with food items and prices
- 🛒 **Shopping Cart** - Add/remove items and manage your order
- 💳 **Secure Checkout** - Easy and secure payment processing
- 📍 **Real-time Tracking** - Track your order from restaurant to doorstep
- 👤 **User Accounts** - Create accounts and save delivery addresses
- ⭐ **Ratings & Reviews** - Rate restaurants and read customer reviews
- 📱 **Responsive Design** - Works seamlessly on all devices

## 💻 Installation

### Prerequisites
- Node.js (v14+)
- Python (v3.7+)
- Git

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/Koustav41/Food-delivery-website.git
   cd Food-delivery-website
   ```

2. **Install Node.js dependencies**
   ```bash
   npm install
   ```

3. **Install Python dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Configure environment variables**
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```

5. **Start the application**
   ```bash
   # Terminal 1: Start Python backend
   python app.py
   
   # Terminal 2: Start frontend development server
   npm start
   ```

6. **Open your browser**
   ```
   Navigate to http://localhost:3000
   ```

## 🚀 Usage

### For Users
1. **Sign Up/Login** - Create an account or log in
2. **Search Restaurants** - Find restaurants by cuisine, rating, or location
3. **Browse Menu** - View items and add to cart
4. **Checkout** - Enter delivery address and payment details
5. **Track Order** - Monitor your delivery in real-time

### For Administrators
1. Access the admin panel at `/admin`
2. Manage restaurants and menus
3. View and manage orders
4. Monitor deliveries
